# URL Shortener 🚀

A simple and efficient URL shortener built with Flask, SQLite, and Bootstrap.

## Features
- 🔗 Shorten long URLs
- 📋 Copy shortened links with one click
- 🔄 Refresh button to reset input
- 💾 Uses SQLite for storage
- 🎨 Clean and responsive UI
